# CGPA Calculator

A simple C++ program that calculates CGPA based on user input.

## 🚀 Features
- Takes subject grades as input
- Calculates overall CGPA
- Simple and easy to use

## 🛠️ Technologies Used
- C++

## ▶️ How to Run
1. Open terminal
2. Compile the code:
   g++ src/cgpa_calculator.cpp -o cgpa
3. Run the program:
   ./cgpa
